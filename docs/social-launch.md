# Cascade Social Launch Copy

## LinkedIn

Built `Cascade` for the Gemini hackathon.

It takes a public GitHub repo plus customer feedback, meeting notes, or a direct feature request and turns that into a visible implementation mission.

Instead of hiding everything behind terminal logs, Cascade shows PM, Architect, Executor, and QA agents moving through the repo, then packages proof with changed files, checks, blockers, and a Continue Working export.

Tech stack:

- Gemini for analysis and execution planning
- React + TypeScript frontend
- Express + TypeScript backend
- Cloud Run deployment target
