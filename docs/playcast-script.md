# Cascade Playcast Script

## 60-second structure

### 0-5s

"Most coding agents help after you already know what to build. Cascade takes a repo and messy product signal, chooses the mission, runs it, and shows proof."

### 5-15s

- Paste a public GitHub repo or file URL.
- Paste customer feedback or a direct feature request.
- Click `Analyze Repo`.

### 15-25s

- Show the selected objective.
- Highlight impacted areas and candidate features.
- Mention that the model is grounding the mission in the repo structure.

### 25-45s

- Launch the mission.
- Let the mission map animate through PM, Architect, Executor, and QA.
- Point out that Cascade shows where the mission is moving instead of streaming a raw terminal transcript.

### 45-60s

- Land on the proof bundle.
- Show changed files.
- Show verification status.
- Open `Continue Working`.
- End on the line: "Cascade turns repo intent into a visible mission."
